# Random-forest-house-price-prediction
House price prediction system built using Random Forest regression with hyperparameter tuning and cross-validation for accurate real estate price estimation.
House Price Prediction Using Random Forest Regression

Overview:

This project builds a machine learning model to predict house prices using the Random Forest regression algorithm. The notebook covers data preprocessing, exploratory data analysis, model training, evaluation, and hyperparameter tuning.

Tools Used:

1. Python

2. Pandas

3. NumPy

4. Matplotlib

5. Seaborn

6. Scikit-learn

7. Joblib


Workflow:

1) Load and explore the dataset

2) Clean and prepare the data

3) Train a Random Forest regression model

4) Evaluate model performance

5) Tune model using RandomizedSearchCV

6) Predict house prices

7) Save trained model

Algorithm: Random Forest Regressor
